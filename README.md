dotconsole
==========

This program provides an interactive editor for Graphviz.

Dependencies
------------

    apt-get install libgtk2-perl graphviz

Installation
------------

To install, run the following commands:

    perl Makefile.PL
    make
    make install

Documentation
-------------

After installing, you can find documentation with the perldoc command.

    perldoc dotconsole
